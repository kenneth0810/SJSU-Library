About SJSUL

Project Overview

Your project is to design an access control schema for a fictitious library (SJSU_Library (SJSUL)) and to implement a simple web-based system to perform primary user enrollment, authentication, and authorization functions for SJSUL.

You may use Python, Java, C, or GoLang. If you want to use another programming language, the professor must give you prior approval. Python has valuable libraries and tools. You may use any framework such as Django or Flask. You must provide citations and references for the framework used.

You may use ChatGPT or other AI tools, but you must provide citations and references for the tools and prompts you used. If you use an AI tool, you must enumerate how you used the tool. Many Large Learning Models and AI tools are reportedly prone to errors.

You may store the data using any database technology or store it in "flat files" using formats such as JSON or XML. You must submit a copy of your source code; please ensure you comment your code well and use sound software design principles.

Purpose of Course Project

  • Develop an understanding of the various access control methodologies and apply those methodologies to a design.
  • Build an understanding of the various use cases in an organization and identify how the different access control methods impact and influence the use cases.
  • Understand the hashing techniques available and how they may be used.
  • Implement hashing functions and access control mechanisms for users.
